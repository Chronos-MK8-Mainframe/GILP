# Research Publication & Social Media Guide

## 1. Where to Upload the Paper (Preprints)
Since this is a new finding, you should publish it as a **Preprint**.

1.  **arXiv.org**: The standard for AI/CS research.
    *   *Pros*: High visibility, "serious" academic cred.
    *   *Cons*: May require an "endorsement" from another user if it's your first submission.
    *   *Category*: `cs.AI` (Artificial Intelligence) or `cs.LG` (Machine Learning).
2.  **Zenodo (CERN)**: excellent for open science.
    *   *Pros*: Instant DOI (Digital Object Identifier), accepts code + pdf together.
    *   *Link*: [zenodo.org](https://zenodo.org)
3.  **ViXra.org**: An alternative if arXiv endorsement is difficult to get.
4.  **GitHub**: Simply compiling the PDF and keeping it in your repository is often enough for a portfolio!
